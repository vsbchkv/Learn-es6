There are several types of vehicles. They all have both general parameters (maximum speed, capacity) and individual ones (for example, the body type of a car, the wingspan of an aircraft, the maximum engine power of a boat).
The program downloads a list of vehicles from the server, creates collections of objects, and then displays the data from the collections in tables on the page.

### Install:

Install project dependencies

`yarn`

### Build:

Buld in development mode

`yarn dev`

Buld in development mode with the local server

`yarn start`

Buld for production

`yarn build`

Run eslint to prevent an errors

`yarn js`

Run eslint with autofix errors mode

`yarn js --fix`
